# QuantAI - AI-Powered Stock Analysis Platform

A Next.js application for AI-powered stock analysis with quantitative insights and risk management.

## Features

- Landing page with features showcase
- Dashboard with stock ticker analysis
- Yahoo Finance integration for real-time stock data
- OpenRouter AI for intelligent analysis
- Supabase authentication (placeholder)
- Risk assessment and investment recommendations

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- OpenRouter API

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

Create a `.env.local` file with:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
OPENROUTER_API_KEY=your_openrouter_api_key
```
